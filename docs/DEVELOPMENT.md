# Development Environment Setup

## Prerequisites
- Node.js >= 18
- npm >= 9
- Git

## Quick Start
```bash
git clone https://github.com/maitamdev/uml-gen.git
cd uml-gen
npm install
npm run dev
```

## API Key Setup
1. Get a free API key from Hugging Face
2. Click the settings icon in the app
3. Paste your key and click Save

## Running Tests
```bash
npm run test
```

## Building
```bash
npm run build
```
