🔑 Cấu hình AI Provider

Lấy API Key miễn phí tại huggingface.co/settings/tokens

1
Tạo tài khoản Hugging Face

Truy cập huggingface.co/join → Đăng ký bằng Google hoặc GitHub cho nhanh.

2
Tạo Access Token

Vào Settings → Tokens → Bấm "Create new token"

  • Token name: uml-generator
  • Token type: chọn "Read" (đủ dùng)

Bấm "Create token" → Copy token (bắt đầu bằng hf_...)

3
Dán vào ô trên & Lưu

Dán token hf_... vào ô API Key ở trên → Bấm "Lưu & Kiểm tra". Khi thấy chấm 🟢 xanh là xong!

💡 Miễn phí: 300 requests/giờ — đủ dùng thoải mái!
📖 Click vào đây để xem hướng dẫn setup API Key ↗
